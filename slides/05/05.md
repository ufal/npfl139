# Prioritized Replay

---
# Rainbow DQN Extensions

## Prioritized Replay

Instead of sampling the transitions uniformly from the replay buffer,
we instead prefer those with a large TD error. Therefore, we sample transitions
according to their probability
$$p_t ‚ąĚ \Big|r + ő≥ \max_{a'} Q(s', a'; ‚ÜíőłŐĄ) - Q(s, a; ‚Üíőł)\Big|^ŌČ,$$
where $ŌČ$ controls the shape of the distribution (which is uniform for $ŌČ=0$
and corresponds to TD error for $ŌČ=1$).

New transitions are inserted into the replay buffer with maximum probability
to support exploration of all encountered transitions.

When combined with DDQN, the probabilities are naturally computed as
$$p_t ‚ąĚ \Big|r + ő≥ Q(s', \argmax_{a'}Q(s', a'; ‚Üíőł); ‚ÜíőłŐĄ) - Q(s, a; ‚Üíőł)\Big|^ŌČ,$$

---
# Rainbow DQN Extensions

## Prioritized Replay

Because we now sample transitions according to $p_t$ instead of uniformly,
on-policy distribution and sampling distribution differ. To compensate, we
utilize importance sampling with ratio
$$ŌĀ_t = \left( \frac{1/N}{p_t} \right) ^ő≤.$$

Because the importance sampling ratios $ŌĀ$ can be quite large, the authors
normalize them, as they say ‚Äúfor stability reasons‚ÄĚ, in every batch:
$$ŌĀ_t / \max_{t'‚ąą\textit{batch}} ŌĀ_{t'}.$$

Therefore, the largest normalized importance sampling ratio in every batch is 1.
The fact that normalization should happen in every batch is not explicitly
stated in the paper, and implementations normalizing over the whole replay
buffer also exist; but the DeepMind reference implementation does normalize
batch-wise.

## Multi-step DQN

Instead of Q-learning, we use $n$-step variant of Q-learning, which estimates
return as
$$‚ąĎ_{i=1}^n ő≥^{i-1} R_i + ő≥^n \max_{a'} Q(s', a'; ‚ÜíőłŐĄ).$$

# Noisy Nets

---
# Rainbow DQN Extensions

## Noisy Nets

Noisy Nets are neural networks whose weights and biases are perturbed by
a parametric function of a noise.

~~~
The parameters $‚Üíőł$ of a regular neural network are in Noisy nets represented as
$$‚Üíőł ‚Čą ‚Üíőľ + ‚ÜíŌÉ ‚äô ‚ÜíőĶ,$$
where $‚ÜíőĶ$ is zero-mean noise with fixed statistics. We therefore learn the
parameters $(‚Üíőľ, ‚ÜíŌÉ)$.

~~~
A fully connected layer $‚Üíy = ‚Üíw ‚Üíx + ‚Üíb$ with parameters $(‚Üíw, ‚Üíb)$ is
represented in the following way in Noisy nets:
$$‚Üíy = (‚Üíőľ_w + ‚ÜíŌÉ_w ‚äô ‚ÜíőĶ_w) ‚Üíx + (‚Üíőľ_b + ‚ÜíŌÉ_b ‚äô ‚ÜíőĶ_b).$$

~~~
Each $ŌÉ_{i,j}$ is initialized to $\frac{ŌÉ_0}{\sqrt{n}}$, where $n$ is the number
of input neurons of the layer in question, and $ŌÉ_0$ is a hyperparameter; commonly 0.5.

---
# Rainbow DQN Extensions

## Noisy Nets

The noise $őĶ$ can be for example independent Gaussian noise. However, for
performance reasons, factorized Gaussian noise is used to generate a matrix of
noise. If $őĶ_{i, j}$ is noise corresponding to a layer with $n$ inputs and $m$
outputs, we generate independent noise $őĶ_i$ for input neurons, independent
noise $őĶ_j$ for output neurons, and set
$$őĶ_{i,j} = f(őĶ_i) f(őĶ_j)~~~\textrm{for}~~~f(x) = \operatorname{sign}(x) \sqrt{|x|}.$$
~~~
The authors generate noise samples for every batch, sharing the noise for all
batch instances (consequently, during loss computation, online and target
network use independent noise).

~~~
### Deep Q Networks
When training a DQN, $őĶ$-greedy is no longer used (all policies are greedy), and
all fully connected layers are parametrized as noisy nets in both the current
and target network (i.e., networks produce samples from the distribution of
returns, and greedy actions still explore).

## Distributional RL: C51

Instead of an expected return $Q(s, a)$, we could estimate the distribution of
expected returns $Z(s, a)$‚ÄĒthe _value distribution_.

~~~
The authors define the distributional Bellman operator $ūĚď£^ŌÄ$ as:
$$ūĚď£^ŌÄ Z(s, a) ‚ČĚ R(s, a) + ő≥ Z(S', A')~~~\textrm{for}~~~S'‚ąľp(s, a), A'‚ąľŌÄ(S').$$

~~~
The authors of the paper prove similar properties of the distributional Bellman
operator compared to the regular Bellman operator, mainly being a contraction
under a suitable metric.

---
# Rainbow DQN Extensions

## Distributional RL: C51

The distribution of returns is modeled as a discrete distribution parametrized
by the number of atoms $N ‚ąą ‚Ąē$ and by $V_\textrm{MIN}, V_\textrm{MAX} ‚ąą ‚ĄĚ$.

~~~
Support of the distribution are atoms
$$\{z_i ‚ČĚ V_\textrm{MIN} + i őĒz : 0 ‚Č§ i < N\}\textrm{~~~for~}őĒz ‚ČĚ \frac{V_\textrm{MAX} - V_\textrm{MIN}}{N-1}.$$

~~~
The atom probabilities are predicted using a $\softmax$ distribution as
$$Z_{‚Üíőł}(s, a) = \left\{z_i\textrm{ with probability }p_i = \frac{e^{f_i(s, a; ‚Üíőł)}}{‚ąĎ_j e^{f_j(s, a; ‚Üíőł)}}\right\}.$$

---
# Rainbow DQN Extensions

## Distributional RL: C51

![w=30%,f=right](dqn_distributional_operator.svgz)

After the Bellman update, the support of the distribution $R(s, a) + ő≥Z(s', a')$
is not the same as the original support. We therefore project it to the original
support by proportionally mapping each atom of the Bellman update to immediate
neighbors in the original support.

~~~
$$ő¶\big(R(s, a) + ő≥Z(s', a')\big)_i ‚ČĚ
  ‚ąĎ_{j=1}^N \left[ 1 - \frac{\left|[r + ő≥z_j]_{V_\textrm{MIN}}^{V_\textrm{MAX}}-z_i\right|}{őĒz} \right]_0^1 p_j(s', a').$$

~~~
The network is trained to minimize the Kullback-Leibler divergence between the
current distribution and the (mapped) distribution of the one-step update
$$D_\textrm{KL}\Big(ő¶\big(R + ő≥Z_{‚ÜíőłŐĄ}\big(s', \argmax_{a'} ūĚĒľZ_{‚ÜíőłŐĄ}(s', a')\big)\big) \Big\| Z_{‚Üíőł}\big(s, a\big)\Big).$$

# Rainbow Architecture

Rainbow combines all described DQN extensions. Instead of $1$-step updates,
$n$-step updates are utilized, and KL divergence of the current and target
return distribution is minimized:
$$D_\textrm{KL}\Big(ő¶\big({\textstyle ‚ąĎ}_{i=0}^{n-1} ő≥^i R_{t+i+1} + ő≥^n Z_{‚ÜíőłŐĄ}\big(S_{t+n}, \argmax_{a'} ūĚĒľZ_{‚Üíőł}(S_{t+n}, a')\big)\big) \Big\| Z(S_t, A_t)\Big).$$

~~~
The prioritized replay chooses transitions according to the probability
$$p_t ‚ąĚ D_\textrm{KL}\Big(ő¶\big({\textstyle ‚ąĎ}_{i=0}^{n-1} ő≥^i R_{t+i+1} + ő≥^n Z_{‚ÜíőłŐĄ}\big(S_{t+n}, \argmax_{a'} ūĚĒľZ_{‚Üíőł}(S_{t+n}, a')\big)\big) \Big\| Z(S_t, A_t)\Big)^w.$$

~~~
Network utilizes dueling architecture feeding the shared representation $f(s; ő∂)$
into value computation $V(f(s; ő∂); ő∑)$ and advantage computation $A_i(f(s; ő∂), a; Ōą)$ for atom $z_i$,
and the final probability of atom $z_i$ in state $s$ and action $a$ is computed as
$$p_i(s, a) ‚ČĚ
  \frac{e^{V_i(f(s; ő∂); ő∑) + A_i(f(s; ő∂), a; Ōą) - \sum_{a' ‚ąą ūĚďź} A_i(f(s; ő∂), a'; Ōą)/|ūĚďź|}}
  {\sum_j e^{V_j(f(s; ő∂); ő∑) + A_j(f(s; ő∂), a; Ōą) - \sum_{a' ‚ąą ūĚďź} A_j(f(s; ő∂), a'; Ōą)/|ūĚďź|}}.$$

# Contraction in C51

In C51, instead of an expected return $Q(s, a)$, we estimate the distribution of
expected returns $Z(s, a)$, and employ the distributional Bellman operator $ūĚď£^ŌÄ$:
$$ūĚď£^ŌÄ Z(s, a) ‚ČĚ R(s, a) + ő≥ Z(S', A')~~~\textrm{for}~~~S'‚ąľp(s, a), A'‚ąľŌÄ(S').$$

~~~
The authors of the paper prove similar properties of the distributional Bellman
operator compared to the regular Bellman operator, mainly being a contraction
under a suitable metric: the **Wasserstein metric** $W_p$ over distributions.

~~~
- For Wasserstein metric $W_p$, the authors define  
  $$WŐĄ_p(Z_1, Z_2)‚ČĚ\sup\nolimits_{s, a} W_p\big(Z_1(s, a), Z_2(s, a)\big)$$
  and prove that $ūĚď£^ŌÄ$ is a ő≥-contraction in $WŐĄ_p$.

~~~
- However, $ūĚď£^ŌÄ$ is not a contraction in KL divergence nor in total variation
  distance.

---
# Wasserstein Metric $W_1$

For two probability distributions over reals with cumulative density functions
(CDFs) $F$ and $G$, the 1-Wasserstein metric $W_1$ is
$$W_1(őľ, őĹ) = ‚ąę\nolimits_0^1 \big|F^{-1}(q) - G^{-1}(q)\big| \d q.$$
where the inverse CDFs $F^{-1}$ and $G^{-1}$ are called **quantile functions**.

# General Wasserstein Metric $W_p$

For two probability distributions $őľ, őĹ$ on a metric space with metric $d$,
Wasserstein metric $W_p$ is defined as
$$W_p(őľ, őĹ) ‚ČĚ \inf_{ő≥‚ąąőď(őľ,őĹ)} \Big(ūĚĒľ_{(x, y)‚ąľő≥} d\big(x, y\big)^p\Big)^{1/p},$$
~~~
where $őď(őľ,őĹ)$ is a set of all _couplings_, each being a joint probability
distribution whose marginals are $őľ$ and $őĹ$, respectively.
~~~
A possible intuition is the optimal transport of probability mass from $őľ$ to
$őĹ$.

~~~
For distributions over reals with CDFs $F, G$, the optimal transport has an
analytic solution:

![w=27.5%,f=right](wasserstein-1.svgz)

$$W_p(őľ, őĹ) = \bigg(‚ąę\nolimits_0^1 \big|F^{-1}(q) - G^{-1}(q)\big|^p \d q\bigg)^{1/p},$$
where $F^{-1}$ and $G^{-1}$ are _quantile functions_, i.e., inverse CDFs.

~~~
For $p=1$, the 1-Wasserstein metric corresponds to area ‚Äúbetween‚ÄĚ F and G, and
in that case we can compute it also as $W_1(őľ, őĹ) = ‚ąę\nolimits_x \big|F(x)- G(x)\big| \d x.$

# Distributional RL with Quantile Regression

The same authors later proposed a different approach, which actually manages to minimize
the 1-Wasserstein distance.

~~~
In contrast to C51, where $Z(s, a)$ is represented using a discrete distribution
on a fixed ‚Äúcomb‚ÄĚ support of uniformly spaced locations, we now represent it
as a _quantile distribution_‚ÄĒas quantiles $őł_i(s, a)$ for fixed
probabilities $ŌĄ_1, ‚Ä¶, ŌĄ_N$ with $ŌĄ_i = \frac{i}{N}$.

~~~
![w=37%,f=right](qr_dqn_1wasserstein.svgz)

Formally, we can define the quantile distribution as a uniform combination of
$N$ Diracs:
$$Z_őł(s, a) ‚ČĚ \frac{1}{N} ‚ąĎ_{i=1}^N őī_{őł_i(s, a)},$$
~~~
so that the cumulative density function is a step function increasing by
$\frac{1}{N}$ on every quantile $őł_i$.

---
# Distributional RL with Quantile Regression

The quantile distribution offers several advantages:

~~~
- a fixed support is no longer required;

~~~
- the projection step $ő¶$ is not longer needed;

~~~
- this parametrization enables direct minimization of the Wasserstein loss.

# Distributional RL with Quantile Regression

Recall that 1-Wasserstein distance between two distributions $őľ, őĹ$ can be computed as
$$W_1(őľ, őĹ) = ‚ąę\nolimits_0^1 \big|F_őľ^{-1}(q) - F_őĹ^{-1}(q)\big| \d q,$$
where $F_őľ$, $F_őĹ$ are their cumulative density functions.

~~~
For arbitrary distribution $Z$, we denote the most accurate quantile
distribution as
$$ő†_{W_1} Z ‚ČĚ \argmin_{Z_őł} W_1(Z, Z_őł).$$

~~~
In this case, the 1-Wasserstein distance can be written as
$$W_1(Z, Z_őł) = ‚ąĎ_{i=1}^N ‚ąę\nolimits_{ŌĄ_{i-1}}^{ŌĄ_i} \big|F_Z^{-1}(q) - őł_i\big| \d q.$$

It can be proven that for continuous $F_Z^{-1}$, $W_1(Z, Z_őł)$ is minimized by
(for proof, see Lemma¬†2 of Dabney et al.: Distributional Reinforcement Learning
with Quantile Regression, or consider how the 1-Wasserstein distance changes in

$$\bigg\{őł_i ‚ąą ‚ĄĚ \bigg| F_Z(őł_i) = \frac{ŌĄ_{i-1} + ŌĄ_i}{2}\bigg\}.$$

~~~
We denote the _quantile midpoints_ as
$$ŌĄŐā_i ‚ČĚ \frac{ŌĄ_{i-1} + ŌĄ_i}{2}.$$

~~~
In the paper, the authors prove that the composition
$ő†_{W_1} ūĚď£^ŌÄ$ is ő≥-contraction in $WŐĄ_‚ąě$, so repeated

# Quantile Regression

Our goal is now to show that it is possible to estimate a quantile $ŌĄ ‚ąą [0, 1]$
by minimizing a loss suitable for SGD.

~~~
Assume we have samples from a distribution $P$.

~~~
- Minimizing the MSE of $xŐā$ and the samples of $P$,
  $$xŐÉ = \argmin\nolimits_xŐā\, ūĚĒľ_{x‚ąľP} \big[(x - xŐā)^2\big],$$
  yields the _mean_ of the distribution, $xŐÉ = ūĚĒľ_{x‚ąľP}[x]$.

~~~
  To show that this holds, we compute the derivative of the loss with respect to
  $xŐā$ and set it to 0, arriving at
  $$0 = ūĚĒľ_x [2(xŐā - x)] = 2 ūĚĒľ_x[xŐā] - 2ūĚĒľ_x[x] = 2\big(xŐā - ūĚĒľ_x[x]\big).$$

# Quantile Regression

Assume we have samples from a distribution $P$ with cumulative density function
$F_P$.

- Minimizing the mean absolute error (MAE) of $xŐā$ and the samples of $P$,
  $$xŐÉ = \argmin\nolimits_xŐā\, ūĚĒľ_{x‚ąľP} \big[|x - xŐā|\big],$$
~~~
  yields the _median_ of the distribution, $xŐÉ = F_P^{-1}(0.5)$.

~~~
  We prove this again by computing the derivative with respect to $xŐā$, assuming
  the functions are nice enough that the Leibniz integral rule can be used:

~~~
  $\displaystyle \frac{‚ąā}{‚ąāxŐā} ‚ąę_{-‚ąě}^{‚ąě} P(x) |x - xŐā| \d x = \frac{‚ąā}{‚ąāxŐā} \bigg[‚ąę_{-‚ąě}^{xŐā} P(x) (xŐā - x) \d x  + ‚ąę_xŐā^‚ąě P(x) (x - xŐā) \d x \bigg]$

~~~
  $\displaystyle \hphantom{\frac{‚ąā}{‚ąāxŐā} ‚ąę_{-‚ąě}^{‚ąě} P(x) |xŐā - x| \d x} = ‚ąę_{-‚ąě}^{xŐā} P(x) \d x - ‚ąę_xŐā^‚ąě P(x) \d x$

~~~
  $\displaystyle \hphantom{\frac{‚ąā}{‚ąāxŐā} ‚ąę_{-‚ąě}^{‚ąě} P(x) |xŐā - x| \d x} = 2 ‚ąę_{-‚ąě}^{xŐā} P(x) \d x - 1 = 2 F_P(xŐā) - 1 = 2 \big(F_P(xŐā) - \tfrac{1}{2}\big).$

# Leibniz integral rule

The Leibniz integral rule for differentiation under the integral sign states that
for $-‚ąě < a(x), b(x) < ‚ąě$,

$$\frac{‚ąā}{‚ąā x} \bigg[‚ąę_{a(x)}^{b(x)} f(x, t) \d t \bigg] =$$

~~~
$$ = ‚ąę_{a(x)}^{b(x)} \frac{‚ąā}{‚ąā x} f(x, t) \d t
   + \bigg(\frac{‚ąā}{‚ąā x} b(x)\bigg) f\big(x, b(x)\big)
   - \bigg(\frac{‚ąā}{‚ąā x} a(x)\bigg) f\big(x, a(x)\big).$$

~~~
_Sufficient condition for the Leibniz integral rule to hold is that the $f(x,
y)$ and its partial derivative $\frac{‚ąā}{‚ąāx}f(x, y)$ are continuous in both $x$
and $t$, and $a(x)$ and $b(x)$ are continuous and have continuous derivatives._

~~~
_If any of the bounds is improper, additional conditions must hold, notably that
the integral of the partial derivatives of $f$ must converge._

- By generalizing the previous result, we can show that for a quantile $ŌĄ ‚ąą [0,
  1]$, if
  $$xŐÉ = \argmin\nolimits_xŐā\, ūĚĒľ_{x‚ąľP} \big[(x - xŐā)(ŌĄ - [x ‚Č§ xŐā])\big],$$
  then $xŐÉ = F_P^{-1}(ŌĄ)$.
~~~
  Let $ŌĀ_ŌĄ(x - xŐā) ‚ČĚ (x - xŐā)(ŌĄ - [x ‚Č§ xŐā]) = |x - xŐā| ‚čÖ |ŌĄ - [x ‚Č§ xŐā]|$.
~~~
  This loss penalizes overestimation errors with weight $1-ŌĄ$, underestimation
  errors with $ŌĄ$.

~~~
  $\displaystyle \frac{‚ąā}{‚ąāxŐā} ‚ąę_{-‚ąě}^{‚ąě} P(x) (x - xŐā)(ŌĄ - [x ‚Č§ xŐā]) \d x =$

~~~
  $\displaystyle \kern2em = \frac{‚ąā}{‚ąāxŐā} \bigg[(ŌĄ-1) ‚ąę_{-‚ąě}^{xŐā} P(x) (x - xŐā) \d x + ŌĄ ‚ąę_xŐā^‚ąě P(x) (x - xŐā) \d x \bigg]$

~~~
  $\displaystyle \kern2em = (\textcolor{blue}{1} - \textcolor{magenta}{ŌĄ}) ‚ąę_{-‚ąě}^{xŐā} P(x) \d x - \textcolor{magenta}{ŌĄ} ‚ąę_xŐā^‚ąě P(x) \d x = \textcolor{blue}{‚ąę_{-‚ąě}^{xŐā} P(x) \d x} - \textcolor{magenta}{ŌĄ} = F_P(xŐā) - ŌĄ.$

# Quantile Regression

Using the quantile regression, when we have a value distribution $Z$, we can
find the most accurate quantile distribution by minimizing
$$‚ąĎ_{i=1}^N ūĚĒľ_{z ‚ąľ Z} \big[ŌĀ_{ŌĄŐā_i}(z - őł_i)\big].$$

~~~
However, the quantile loss is not smooth around zero, which could limit
performance when training a model. The authors therefore propose the
**quantile Huber loss**, which acts as an asymmetric squared loss
in interval $[-őļ, őļ]$ and fall backs to the standard quantile loss outside this
range.

~~~
Specifically, let
$$ŌĀ_ŌĄ^őļ(z - őł) ‚ČĚ \begin{cases}
  \big|ŌĄ - [z ‚Č§ őł]\big| ‚čÖ \tfrac{1}{2}\big(z - őł\big)^2 &~~\textrm{if}~~ |z - őł| ‚Č§ őļ,\\
  \big|ŌĄ - [z ‚Č§ őł]\big| ‚čÖ őļ\big(|z - őł| - \tfrac{1}{2}őļ\big) &~~\textrm{otherwise},\\
  \end{cases}$$
with a special case of plain quantile loss for $őļ=0$, i.e., $ŌĀ_ŌĄ^0(z - őł) ‚ČĚ |ŌĄ-[z‚Č§őł]|‚čÖ|z - őł|$.

# Implicit Quantile Networks for Distributional RL

In IQN (implicit quantile regression), the authors (again the same team as in
C51 and DR-DQN) generalize the value distribution representation to predict
_any given quantile $ŌĄ$_.

![w=69%,f=right](iqn_architecture_comparison.svgz)

~~~
- The $Ōą(s)$ is a convolutional stack from DQN, composed of

  - CNN $8√ó8$, stride 4, 32 filters, ReLU;
  - CNN $4√ó4$, stride 2, 64 filters, ReLU;
  - CNN $3√ó3$, stride 1, 64 filters, ReLU.

~~~
- The $f$ is an MLP:
  - fully connected layer with 512 units, ReLU;
  - output layer, 1 unit.

# Implicit Quantile Networks for Distributional RL

The quantile $ŌĄ$ of the value distribution, $Z_ŌĄ(s, a)$, is modeled as
$$Z_ŌĄ(s, a) ‚Čą f\big(Ōą(s) ‚äô ŌÜ(ŌĄ)\big)_a.$$

~~~
- Other ways than multiplicative combinations were tried (concatenation, or
  residual computation $Ōą(s)‚äô(1+ŌÜ(ŌĄ))$), but the multiplicative form delivered
  the best results.

~~~
- The quantile $ŌĄ$ is represented using trainable cosine embeddings with
  dimension $n=64$:
  $$ŌÜ_j(ŌĄ) ‚ČĚ \operatorname{ReLU}\Big(‚ąĎ\nolimits_{i=0}^{n-1} \cos(ŌÄ i ŌĄ) w_{i,j} + b_j\Big).$$

~~~
- The target policy is greedy with respect to action-value approximation
  computed using $K$ samples $ŌĄŐÉ_k ‚ąľ U[0, 1]$:
  $$ŌÄ(x) ‚ČĚ \argmax_a \frac{1}{K} ‚ąĎ_{k=1}^K Z_{ŌĄŐÉ_k}(x, a).$$
~~~
  - As in DQN, the exploration is still performed by using the $őĶ$-greedy
    policy.
